Usage:
  bin2array.exe [options] input_file
Options:
  -h                         display this help screen
  -d                         enable debug print
  -w <width>                 set line width(default 16 byte)
  -l                         enable linux mode(line ends with "\n" instead of "\r\n")
  -buf_name <name>           set global variable name
  -section <name>            set section
  -o <output_file>           set suffix of output file(default "input_file.txt")
